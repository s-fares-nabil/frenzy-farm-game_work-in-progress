# Farm Frenzy Management

A C++ desktop farm simulation game built with Object-Oriented Programming principles and the CMU Graphics Library.

## Features

- 4 interconnected subsystems: inventory, production, in-game economy
- Object-Oriented architecture with clean separation of concerns
- Stable 60 FPS core game loop
- Over 1,200 lines of C++ code

## How to Build

Open `graphics_prj.sln` in Visual Studio and build the solution.

## Requirements

- Visual Studio 2019+
- CMU Graphics Library (included)
